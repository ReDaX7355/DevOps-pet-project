# Пет проект по DevOps

Этот проект предназначен для демонтрации навыков, полученных путем обучения методологии DevOps в системном админитрировании.

## 0. Инициация репозитория
Первоначально добавляю сюда мини проект, созданный с помощью ИИ, для дальнейшей работы с ним. 

#### Описание проекта
Данный проект, это frontend приложение "Todo", который позволяет добавлять задачи в базу данных Postgress через backend NodeJS.

**Состав проекта**
- Frontend: React Vite + Tailwind
- Backend: NodeJS + express
- DB: Postgress

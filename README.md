# Пет проект по DevOps

Этот проект предназначен для демонтрации навыков, полученных путем обучения методологии DevOps в системном администрировании.

## 0. Инициализация репозитория
Первоначально добавляю сюда мини web приложение, созданное с помощью ИИ, для дальнейшей работы с ним.

#### Описание веб приложения
Это frontend приложение "Todo", в котором можно добавлять задачи в базу данных Postgress через backend NodeJS.

**Состав проекта**
- Frontend: React Vite + Tailwind
- Backend: NodeJS + express
- DB: Postgress

## 1. Создание Docker конфигураций
Далее создаю docker файлы для каждого сегмента приложения (Frontend, Backend). Для DB не создается dockerfile, создание контейнера производится в общем конфиге docker-compose.
Он как раз создан для разворачивания всего приложения сразу.

Также на этом этапе были подправлены настройки связи между сегментами приложения, чтобы внутри контейнеров они держали связь друг с другом.
- Frontend: frontend/src/App.tsx указывается переменная API_URL со значением IP адреса backend, порт бэкенда: 4450
- Backend: переменные указанные в docker-compose.yml - DATABASE_URL и PORT, подключение к БД и порт
- DB: рабоатет на стандартном порту 5432
